## 🤖 AI Projects

### Intent-Aware Interoperability Platform
> *Multi-Agent AI System · Healthcare / Life Sciences*

A multi-role AI agent system that resolves the same clinical question differently based on the requesting stakeholder's intent — Clinician, Trial Investigator, Payer, or Regulator.

- 🧠 Routes through a **semantic layer** that infers role and goal from a `POST /intent` API call
- 🏥 Integrates **FHIR, CDISC SDTM, X12 claims**, and real-world evidence benchmarks as live inputs — all derived columns (RECIST response, CTCAE grading, PDC adherence, eGFR decline) computed at runtime for full auditability
- ⚡ **Prompt-engineered LLM chains** fire exactly one clinical alert per evaluation across 22–25 protocol rules per role — production-style model evaluation and comparison workflows
- 📄 **RAG-based semantic retrieval** for payer policy evaluation — retrieves relevant policy PDF clauses by ICD-10/NDC and generates word-to-word audit justification logs for regulatory defensibility

**Stack:** LLMs · Prompt Engineering · RAG · REST APIs · JSON Schema · FHIR · CDISC
